# GrabbingAPI

----
## usage
  * Grabbing a page   
Url: **'/wrappers'**  
Payload: **POST** `{
"page": "http://devpolish.github.io"
}`  

*Automatically it will return the object with tags and their texts.*


  * Index previous links used.

  **GET** `/searches`  
  
  *This will returns all links previous used*
